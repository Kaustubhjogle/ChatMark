# PromptPane Privacy Policy

Effective date: April 17, 2026

PromptPane is a Chrome extension that helps users navigate their own prompts in supported AI chat pages.

## What data PromptPane accesses

On supported chat URLs, PromptPane reads visible page content needed to identify user prompt messages and short snippets of the first following assistant replies for preview.

## What PromptPane does with data

- Processes chat content locally in the browser.
- Uses the data only to render the prompt list, short answer previews, and scroll-to-message navigation.

## What PromptPane does not do

- Does not transmit chat content to external servers.
- Does not store chat content remotely.
- Does not sell personal data.
- Does not use data for advertising.

## Data sharing

PromptPane does not share user data with third parties.

## Data retention

PromptPane does not persist chat content outside the current page context required for operation.

## Security

PromptPane has no remote code loading and no external network requests in normal operation.
